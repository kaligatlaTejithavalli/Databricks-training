# SQL to PySpark ETL Pipeline

## Overview
This project demonstrates an end-to-end ETL pipeline using SQL and PySpark in Databricks.

## Technologies
- PySpark
- SQL
- Databricks
- CSV
- JSON
- Parquet

## Features
- Data Ingestion
- Schema Inspection
- Missing Value Handling
- Data Cleaning
- Filtering
- Aggregations
- Joins
- Window Functions
- ETL Pipeline

## Datasets
- customers.csv (1000 records)
- sales.csv (5000 records)

## Output
- Daily Sales
- City-wise Revenue
- Repeat Customers
- Highest Spending Customer
- Final Reporting Table
