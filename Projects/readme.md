# PySpark Data Engineering Projects

---

## Overview

This repository contains a collection of hands-on PySpark projects developed to strengthen core Data Engineering and Big Data processing skills. The projects focus on real-world data processing tasks, including data cleaning, SQL-to-PySpark transformations, analytics, window functions, ETL workflows, and business reporting using Apache Spark.

These projects were built using **Python**, **PySpark**, **Spark SQL**, and **Google Colab** to simulate practical data engineering scenarios and demonstrate scalable data processing techniques.

---

## Project 1 – Employee Salary Analytics

### Objective

Perform employee salary analysis using PySpark DataFrames and Spark SQL while learning aggregations, filtering, window functions, and business reporting techniques.

### Features

- Data Loading
- Data Cleaning
- Highest Paid Employees by Department
- Average Salary by City
- Salary Band Classification
- Year-wise Hiring Analysis
- Employees Earning Above Department Average
- Window Functions
- Spark SQL Queries

### PySpark Concepts

- DataFrames
- GroupBy
- Aggregations
- Window Functions
- Conditional Columns
- Joins
- Date Functions
